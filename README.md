# Coefficienti di ammortamento

Tabella in formato JSON con i coefficienti ministeriali di ammortamento.

I coefficienti di ammortamento sono stabiliti dal D.M. del Ministero delle Finanze del 31 dicembre 1988 e suddivisi in base all'attività (Gruppo/Specie) e alla tipologia di bene strumentale.

## Contenuto del repository

- `coeff_ammortamento.json` — la tabella dei coefficienti in formato JSON.
- `index.html` — un visualizzatore HTML per consultare la tabella.
